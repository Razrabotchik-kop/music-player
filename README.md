# 🎵 Music Player

[![Version](https://img.shields.io/badge/version-1.2.0-blue.svg)](https://github.com/YOUR_USERNAME/music-player/releases)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

Стильный музыкальный плеер для Linux с тёмной темой.

## ✨ Возможности

- 🎨 Тёмный современный интерфейс
- 💾 Автосохранение плейлиста
- ⌨️ Горячие клавиши (←→ громкость, ↑↓ треки)
- 🔄 Автопереключение треков
- 🔊 Регулировка громкости

## 📥 Установка

```bash
# Скачать DEB пакет из раздела Releases
# Или собрать самостоятельно:

git clone https://github.com/YOUR_USERNAME/music-player.git
cd music-player
chmod +x build.sh
./build.sh
sudo dpkg -i music-player_*.deb
sudo apt install -f
